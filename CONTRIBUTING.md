# Contributing to Temporal Causality Engine

Thanks for your interest in contributing.

This project is research-oriented and prioritizes **statistical correctness,
reproducibility, and methodological transparency** over convenience or visual appeal.

Please read the guidelines below before proposing changes.

---

## Development setup

### Requirements
- Python **>= 3.10**
- Conda or virtualenv
- pip

### Recommended setup (Conda)
```bash
conda env create -f environment.yml
conda activate temporal-causality-engine
pip install -e .
